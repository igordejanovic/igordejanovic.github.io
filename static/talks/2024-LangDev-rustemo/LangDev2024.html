Live demo 1 - arithmetic expressions

Why Arithmetic DSL?

  • Universally Understood: Everyone can grasp basic arithmetic, so no need to explain domain concepts.
  • Clear Ambiguity: The precedence ambiguity is simple yet powerful enough to showcase GLR parsing.
  • Extendable: We can extend the language with functions (e.g., sin, cos, log) or more complex expressions. This is left as an excercise.

Step 1 - create Rust project

cargo new --bin arithmetic

The layout of the generated project is: 

arithmetic
├── Cargo.toml
└── src
    └── main.rs

Step 2 - Create expression grammar

  • File src/arithmetic.rustemo 

    E: E '+' E
 | E '*' E
 | Number
;

terminals
Number: /\d+/;
Add: '+';
Mul: '*';
